# CSV 数据导入/导出格式说明

College Tracker 支持通过 CSV 文件批量导入和导出课程、经历与荣誉数据。

---

## 通用规则

- 文件编码为 **UTF-8**。
- **第一行必须是表头**（列名），格式必须与下方模板完全一致（包括标点符号）。
- 数据从第二行开始，每行一条记录。
- 包含逗号 `,`、双引号 `"` 或换行的字段需要用双引号包裹，字段内的双引号需写为两个连续的双引号 `""`。
- 空行会自动跳过。
- 导出时程序会自动添加 UTF-8 BOM，确保在 Excel 中正确显示中文。

---

## 1. 课程 (Courses)

### 表头

```
课程名称,学分,成绩,学期,核心课程
```

### 字段说明

| 列名 | 说明 | 是否必填 | 示例 |
|------|------|---------|------|
| 课程名称 | 课程名字 | 是 | 高等数学 |
| 学分 | 数值，必须大于 0 | 是 | 4 或 3.5 |
| 成绩 | 数值，范围 0–100 | 是 | 92 |
| 学期 | 必须为以下值之一（见下方列表） | 是 | 大一上 |
| 核心课程 | 是否在简历中作为核心课程展示，填写 `是` 或 `否` | 否 | 是 |

**学期可选值：** `大一上` `大一下` `大二上` `大二下` `大三上` `大三下` `大四上` `大四下`

> 导入时成绩会自动换算为 GPA（绩点），无需手动填写。
>
> 为兼容旧文件，缺少“核心课程”列的四列课程 CSV 仍可导入，
> 导入后会默认标记为“否”。导出会统一使用新的五列格式。

### 示例

```csv
课程名称,学分,成绩,学期,核心课程
高等数学,4,92,大一上,是
线性代数,3,85,大一上,是
大学英语,3,78,大一下,否
程序设计基础,3.5,88,大一下,否
```

## 2. 经历 (Experiences)

> 在"经历 & 荣誉"页面中切换到"课外活动"标签后，导入/导出操作针对此表。

### 表头

```
标题,类型,时间,描述
```

### 字段说明

| 列名 | 说明 | 是否必填 | 示例 |
|------|------|---------|------|
| 标题 | 经历标题 | 是 | 数学建模竞赛 |
| 类型 | 必须为以下值之一：`实习` `竞赛` `项目` `其他` | 是 | 竞赛 |
| 时间 | 日期，建议格式 yyyy-MM-dd | 否 | 2024-03-15 |
| 描述 | 详细描述内容 | 否 | 获得省级一等奖 |

### 示例

```csv
标题,类型,时间,描述
数学建模竞赛,竞赛,2024-03-15,获得省级一等奖
暑期互联网实习,实习,2024-07-01,在XX公司担任后端开发实习生
学生会活动策划,项目,2023-09-10,负责校园文化节整体策划与执行
实验室科研助理,其他,2024-02-20,参与自然语言处理课题研究
```

## 3. 荣誉 (Awards)

> 在"经历 & 荣誉"页面中切换到"个人荣誉"标签后，导入/导出操作针对此表。

### 表头

```
奖项名称,荣誉级别,获奖时间,奖金金额
```

### 字段说明

| 列名 | 说明 | 是否必填 | 示例 |
|------|------|---------|------|
| 奖项名称 | 荣誉/奖项名称 | 是 | 国家奖学金 |
| 荣誉级别 | 必须为以下值之一：`国家级` `省级` `校级` `院级` | 是 | 国家级 |
| 获奖时间 | 日期，建议格式 yyyy-MM-dd | 否 | 2024-09-01 |
| 奖金金额 | 整数数值，无则为 0 | 否 | 8000 |

### 示例

```csv
奖项名称,荣誉级别,获奖时间,奖金金额
国家奖学金,国家级,2024-09-01,8000
优秀学生干部,校级,2024-06-15,500
ACM程序设计竞赛一等奖,省级,2024-04-20,2000
学院优秀团员,院级,2023-12-10,0
```

## 操作步骤

### 导入数据

1. 按照上方模板准备 CSV 文件（确保用 UTF-8 编码保存）。
2. 打开 College Tracker，进入对应页面（课程 / 经历 & 荣誉）。
3. 点击 **"导入CSV"** 按钮。
4. 选择你准备好的 CSV 文件。
5. 程序会显示导入结果（成功条数 / 失败条数）。

### 导出数据

1. 进入对应页面（课程 / 经历 & 荣誉）。
2. 点击 **"导出CSV"** 按钮。
3. 选择保存位置和文件名。
4. 导出完成后会显示导出的数据条数。

## 常见问题

**Q: 为什么导出的 CSV 文件在 Excel 中打开时中文乱码？**

A: 导出文件已包含 UTF-8 BOM 标记。在 Excel 中打开时，请使用"数据" → "从文本/CSV 导入"，文件原始格式选择 "UTF-8" 即可正确显示中文。也可以直接用 WPS、Numbers 或记事本打开，它们通常能自动识别编码。

**Q: 导入时提示"CSV 表头不匹配"怎么办？**

A: 请检查 CSV 文件第一行的列名是否与本文档中的表头模板**完全一致**（包括中文标点）。注意不要有多余的空格或列。

**Q: 导入时部分数据失败怎么办？**

A: 导入完成后会显示成功和失败的数量。常见失败原因：
- 课程：学分 ≤ 0 或不是数字、成绩超出 0–100、学期不在允许列表中、“核心课程”不是 `是/否`
- 经历：类型不在 `实习/竞赛/项目/其他` 范围内
- 荣誉：荣誉级别不在 `国家级/省级/校级/院级` 范围内

请根据提示检查 CSV 文件中对应行的数据。
