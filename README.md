# CollegeTracker
还在为繁杂的大学生活而规范化记录大学生活轨迹而苦恼吗？
市面上现在缺乏专门记录大学生涯规划的应用，现有的软件（Excel、备忘录等）缺乏与大学生的关联性与统计能力

为此，我们开发了一款基于 Qt 的轻量桌面应用程序，学生进行学业成绩管理以及个人生活（实习、竞赛、荣誉等）追踪，为未来保研、留学或考研提供数据支持。

本项目具有：
* 完善的数据追踪功能：包括学生成绩追踪、学生荣誉最终等
* 同时支持数据的导入导出
* 用户可以根据导入的数据生成一份简历，并且有多种模板可供选择


本项目现已支持支持多平台：Linux、Windows、macOS
